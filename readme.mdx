## Ignite Lab - Design System

Projeto de um Design System criado durante o Ignite Lab da Rocketseat. Desenvolvido com o intuito de aperfeiçoar
os conhecimentos com React e principalmente com o Storybook e Figma.

## Tecnologias

- [React](https://reactjs.org/) - Biblioteca JavaScript para criar interfaces de usuário.
- [Storybook](https://storybook.js.org/) - Biblioteca para desenvolvimento de componentes isolados.
- [Tailwind CSS](https://tailwindcss.com/) - Framework CSS para estilização de componentes.
- [Figma](https://www.figma.com/) - Ferramenta de prototipação de interfaces.
- [Vite](https://vitejs.dev/) - Ferramenta de construção de aplicações web modernas.
- [TypeScript](https://www.typescriptlang.org/) - Superset JavaScript que adiciona tipagem estática.
